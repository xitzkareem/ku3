\section{Provide formal proofs for the following claims}

\subsection{For the sequence $a_1 = 1 , a_n = 2 \cdot a_{n-1} +1, \forall n \ge 2 $ prove that $ a_n = 2^n -1, \forall n \in \mathbb{Z}^+ $ }
Using principles of mathematical induction, we start by showing that the first proposition holds : 
Let $P(n)$ be the predicate $a_n =2^{n} - 1  $
Base case P(1) : 
\begin{equation}
	1= a_1 = 2^1 -1 = 1 (HOLDS )
\end{equation}
Now we try to prove that $P(k) \implies P(k+1)$ is a tautology
\begin{equation}
	P(k) : a_k = 2^k -1
\end{equation}
\begin{equation}
	P(k+1) : a_{k+1} = 2^{k+1} 	-1 
\end{equation}
The next term in the sequence (left hand side) is
$$a_{k+1} =2 \cdot a_{k-1+1} -1 = 2 \cdot a_{k} -1  $$
 substituting (2) in , we get 
\begin{equation}
	a_{k+1} = 2 \cdot (2^k -1) + 1 = 2 ^{k+1} -1 
\end{equation}
We have shown that $a_n =2^{n} - 1  $ holds for all values $n \in \mathbb{N} ^+$


\subsection{Prove that it holds for   $\forall n \in \mathbb{N}^+ $  $ 3 |4^n 	+5 $}

Base case : $P(n_0)$
We start with proving the base case $n_0 = 1$ is true 
\begin{equation}
P(n_0)	: 4^1 +5  = 9
\end{equation} 
\begin{equation}
	3|9 : True 
\end{equation}

induction step : 
\begin{equation}
	P(k) : 3|4^k +5
\end{equation}
let $f(k)=4^k +5$
\begin{equation}
	P(k+1) : 3|4^{k+1} +5
\end{equation}
The main point here is to manipulate $P(k+1)$ into an expression in terms of $P(k)$
\begin{equation}
		P(k+1) : 4^{k+1} +5 = 4 * 4^{k+1} +5 = 4^k+4^k+4^k+4^k +5 = 3 \cdot 4^k +(4^k+5)
\end{equation}
We can write now that $P(k+1) = f(k) + s ,  s\in \mathbb{N} $. We have already proved that 3 divides $f(k)$, if we can prove that 3 also divides s, we can use use the properties of integers and division and say that if 
$a|b \wedge a|c $ then $a|b+c:T $  
$s = 3 \cdot 4^k$ , since s is a multiple of 3, we can easily conclude that s is divisible by 3, hence $3|f(k)+s: T $
